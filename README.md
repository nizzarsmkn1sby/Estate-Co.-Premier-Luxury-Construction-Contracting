# ESTATE & CO. | Premier Luxury Construction & Contracting

A high-end, elegant landing page designed for a premium construction vendor and contractor.

## Design Philosophy
- **Clean & Professional**: Minimalist layout with plenty of white space.
- **Elegant Aesthetics**: A sophisticated color palette of Gold (`#B8860B`), Deep Matte Black (`#111111`), and Clean White.
- **Modern Typography**: Using 'Playfair Display' for headlines and 'Inter' for body text.
- **Transparency & Trust**: Business values are prominently displayed (Metrics section: 100% Transparency, 0 Delays).

## Features
- **Responsive Design**: Works seamlessly on Mobile, Tablet, and Desktop.
- **Glassmorphism Nav**: Sticky navigation bar with a blurred background effect on scroll.
- **Interactive Elements**: Hover effects on cards, portfolio items, and buttons.
- **SEO Optimized**: Includes meta tags and semantic HTML5 structure.

## Tech Stack
- HTML5
- CSS3 (Custom Grid & Flexbox)
- Vanilla JavaScript
